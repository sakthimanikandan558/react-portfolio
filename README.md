## ğŸ“ Customization

To customize this portfolio for your own use:

1. **Update Personal Information**

   - Edit `index.html` - Change the title
   - Edit `src/pages/Hero/Hero.jsx` - Update name, roles, and code snippet
   - Edit `src/pages/About/About.jsx` - Update professional summary

2. **Update Skills**

   - Edit `src/pages/Skills/Skills.jsx` - Add your skill categories
   - Edit `src/components/globe.jsx` - Update technology icons

3. **Update Experience & Projects**

   - Edit `src/pages/Experience/Experience.jsx` - Add your work experience
   - Edit `src/pages/Projects/Projects.jsx` - Add your projects

4. **Update Education**

   - Edit `src/pages/Education/Education.jsx` - Add your educational background

5. **Update Contact Information**
   - Edit `src/pages/Contact/Contact.jsx` - Update email, phone, location
   - Get your Web3Forms access key from [https://web3forms.com](https://web3forms.com)
